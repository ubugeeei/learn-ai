# 11 — ニューロン、層、MLP を訓練する

## この章で作るもの

スカラーの `Value` による自動微分 (automatic differentiation) を使って、
ニューロン、層、多層パーセプトロン (multilayer perceptron)、MSE 損失
(loss)、SGD を実装し、XOR を訓練します。

- ネットワーク: `src/main/scala/learnai/nn/ScalarNetwork.scala`
- 実験: `src/main/scala/learnai/nn/Xor.scala`

## 1 つのニューロン

入力 $\boldsymbol{x}$、重み $\boldsymbol{w}$、バイアス $b$ について:

$$
z=\boldsymbol{w}\cdot\boldsymbol{x}+b
=\sum_i w_i x_i+b
$$

次に活性化関数 $\phi$ を適用します。

$$
y=\phi(z)
$$

```scala
val weightedInputs = weights.zip(inputs).map { case (weight, input) =>
  weight * input
}
activation(Value.sum(weightedInputs) + bias)
```

重みとバイアスは訓練可能な葉です。入力と中間値はそうではありません。

## 活性化関数が必要な理由

線形関数を合成しても線形のままです。

$$
\boldsymbol{W}_2(\boldsymbol{W}_1\boldsymbol{x}+\boldsymbol{b}_1)+\boldsymbol{b}_2
=(\boldsymbol{W}_2\boldsymbol{W}_1)\boldsymbol{x}
+(\boldsymbol{W}_2\boldsymbol{b}_1+\boldsymbol{b}_2)
$$

非線形な活性化関数がなければ、深くしても非線形な決定境界は増えません。

| 活性化関数 | 式 | 性質 |
| --- | --- | --- |
| 線形 | $x$ | 出力スコアと回帰 |
| Tanh | $\tanh x$ | 滑らかな値域 `[-1,1]` |
| ReLU | $\max(0,x)$ | 単純で、正の側の微分が一定 |

現代の Transformer のフィードフォワード層では GELU や SwiGLU がよく
使われますが、非線形性が必要であることは同じです。

## 層と MLP

層は複数のニューロンを並列に評価します。

$$
\boldsymbol{y}=\phi(\boldsymbol{W}\boldsymbol{x}+\boldsymbol{b})
$$

入力幅 $n$、出力幅 $m$ に対して、重みの形状は `[m,n]`、バイアスの形状は
`[m]` です。

XOR モデルは次のとおりです。

```text
input [2]
  -> hidden [4], tanh
  -> output [1], tanh
```

パラメータ数:

- 隠れ層: `4*2 + 4 = 12`。
- 出力層: `1*4 + 1 = 5`。
- 合計: `17`。

パラメータを数えることは、メモリと計算量の見積もりへの第一歩です。

## XOR には非線形性が必要

目標値は、入力の符号が異なるとき正、一致するとき負です。1 本の直線では
2 つのクラスを分離できません。隠れニューロンがいくつかの中間的な境界を
作り、非線形な活性化関数がそれらを曲げ、出力層が組み合わせます。

## 平均二乗誤差

$$
L=\frac{1}{N}\sum_{i=1}^{N}(\hat y_i-y_i)^2
$$

言語モデルは後の章で交差エントロピーを使いますが、訓練ループの構造は
変わりません。

## 1 ステップを追跡する

```scala
val predictions = dataset.map(example => model(example.inputs).head)
val loss = Loss.meanSquaredError(predictions, targets)
loss.backward()
Sgd.step(model.parameters, learningRate)
```

1. 4 つの例すべてを順伝播する。
2. 誤差を 1 つのスカラーにまとめる。
3. 17 個すべてのパラメータへ逆伝播する。
4. パラメータを更新する。
5. 次のステップで新しいグラフを作り直す。

これはフルバッチ訓練です。大きなデータセットではサンプリングした
ミニバッチを使います。

## 初期化

すべての重みがゼロから始まると、ニューロンは等しい勾配 (gradient) を
受け取り、対称なままになります。シード付きのランダム初期化が、それぞれ
に異なる役割を与えます。

Xavier の一様分布は次を使います。

$$
w\sim U\left(
-\sqrt{\frac{6}{n_{in}+n_{out}}},
\sqrt{\frac{6}{n_{in}+n_{out}}}
\right)
$$

これは、層をまたいで活性化のスケールが消失も爆発もしないようにすること
を狙っています。

## 実行する

```console
$ nix develop -c sbt 'runMain learnai.nn.trainXor'
```

シードが固定されているため、初期化と学習曲線は再現可能です。テストは
低い損失と、4 つの例すべてで正しい符号を要求します。

## 実装ウォークスルー

`Neuron.apply` は、パラメータ、データ、自動微分が初めて出会う場所です。
入力幅を確認し、各入力を 1 つの重みと組にして `Value` ノードどうしを
掛け合わせ、バイアスとともに合計し、設定された活性化関数を適用します。

$$
y=\phi\left(b+\sum_i w_i x_i\right)
$$

入力 `[2,-1]`、重み `[0.5,3]`、バイアス `1` の場合、活性化前の値は
`1 + 0.5*2 + 3*(-1) = -1` です。tanh を使うと出力はおよそ `-0.7616` に
なります。活性化前の計算を明示的なまま保つことで、層を積み重ねる前に
1 つのニューロンを電卓と比較できます。

`Layer` はニューロンのベクトルを所有し、同じ入力ベクトルをそれぞれに
通します。`MultiLayerPerceptron` は、1 つのベクトルを連続する層へ順に
畳み込んでいきます。パラメータの列挙は同じ所有権の木構造に従い、重み、
バイアスの順に安定した順序で返します。オプティマイザ (optimizer) と
決定的なテストはこの順序に依存しています。

ランダム構築は、呼び出し側が所有する 1 つの `SplittableRandom` を受け
取ります。ニューロンごとに新しいシードを作ることはありません。したがって
トップレベルのシードが等しければ、同じ決定的な順序で乱数を消費し、
等しいモデルを生成します。

XOR は 1 つの線形境界では分離できません。隠れ層が非線形な特徴を学習し、
出力層がそれらを組み合わせます。訓練は 4 つの点を反復し、新しいスカラー
グラフを構築し、平均二乗誤差を計算し、backward を呼び、すべての
パラメータを更新します。この小さなデータセットは、すべての予測を検査
できるよう意図的に過学習させています。

## テストの読み方

ニューロンのテストは、明示的な重みと手計算できるアフィン値を使います。
等しいシードでの構築は RNG の所有権を検証します。XOR の統合テストは、
損失の大きな減少と 4 つの分類すべての両方を確認します。損失だけなら、
1 点が誤ったままでも下がりうるからです。幅の誤った入力は、部分的な
内積を生成する代わりに、最初のニューロンで失敗しなければなりません。

## デバッグチェックリスト

1. 層全体より先に、1 つのアフィンなニューロンを検証する。
2. 活性化前の値と活性化後の値を分けて出力する。
3. すべてのパラメータが `parameters` にちょうど 1 回現れることを確認
   する。
4. 等しいシードで結果が異なる場合は、乱数の消費順序を比較する。
5. XOR が停滞する場合は、アーキテクチャを変える前に、4 つの予測すべて、
   勾配の大きさ、活性化の飽和、グラフの再構築を調べる。

## 演習

1. 17 個のパラメータを数え直す。
2. 隠れ層の幅 `1`、`2`、`8` を複数のシードで比較する。
3. tanh を ReLU に置き換え、損失曲線を比較する。
4. いくつかの学習率について、損失が `0.02` を下回る最初のステップを
   見つける。
5. 線形な出力を使い、その値域を調べる。
6. AND のデータセットを訓練し、隠れ層が必要かどうかを確かめる。

## 完了基準

- ニューロンを重み、入力、バイアス、活性化関数に分解できる。
- 積み重ねた線形層が 1 つの線形層に潰れる理由を説明できる。
- 層の形状とパラメータ数を計算できる。
- コードの中で順伝播、損失、逆伝播、更新を特定できる。
- 比較のためにシードを固定する理由を説明できる。
- `ScalarNetworkSuite` が通る。
