# カリキュラム

## ステータス凡例

- ✅ 章、実装、検証パスが完成しています。
- 🚧 学習成果は確定していますが、本文または実装が作成中です。
- ⬜ 計画中です。

各パートは、それより前の成果物のみに依存します。任意の拡張は、メインパスを
壊すことなくスキップできます。

✅ は、その章で述べられたスコープが完了していることを意味するのであって、
コンポーネントがプロダクション水準で完成していることや、学習者がプロフェッ
ショナルとしての成熟度に達したことを意味しません。
[プロフェッショナル能力ロードマップ](05-professional-roadmap.md)が、目に見える
メカニズムから運用上の判断力への M0–M4 の橋渡しを追跡します。

## キャップストーン — 実践的なグラウンディング付きエージェント

ローカル文書をインデックスし、引用付きで回答し、明示的に許可された操作
だけを実行できるエージェントを構築します。完了には次が必要です。

- 交換可能なフェイク、ローカル、リモートのモデルプロバイダ
- すべての外部アクションに対するスキーマ検証、タイムアウト、監査イベント
- 固定された評価セットにおける再現可能な品質、レイテンシ、トークンの指標
- プロンプトインジェクションと汚染されたツール出力を対象とする脅威モデル
- 設計、制約、再現手順を記録したシステムカード
